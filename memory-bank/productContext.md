# Product Context — Liquidity Raid Hunter

## Суть проекта

Стратегия на психологии рынка: после сильного нагнетания объёма (памп) происходит сбор ликвидности (снос стопов).

«Токены ходят только из рук в руки» — мы зарабатываем там, где слабые руки теряют.

## Текущая архитектура (v8.24)

- **Сканер** (Python + ccxt + Telegram) — ищет сигналы на Bybit USDT Perpetual
- **Индикатор** (Pine Script) — визуальное подтверждение на TradingView
- Сканер и индикатор должны быть синхронизированы

## Логика входа (v8.24)

- body_pct ≥ 4.5%
- impulse_strength ≥ 1.25
- volume_ratio ≥ 1.9
- prior_volume ≥ 1.35 (фильтр контекста пампа)
- liquidity_grab
- bearish закрытие
- структурный фильтр
- условие D (volume next ≥ 0.6)

## Управление позицией

- TP1 = 1.6R → закрываем 50% + переводим остаток в Breakeven
- TP2 = 3R
- Стоп: структурный + лимит риска 2%

## Цели

- 2–3 качественных сигнала в день
- Сбор чистой статистики
- Постепенное улучшение на основе реальных данных
