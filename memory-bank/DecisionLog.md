# Decision Log — Liquidity Raid Hunter

## 16.07.2026 — Создание multi-agent системы

**Решение:**  
Отказаться от тяжёлой реализации через CrewAI на текущем этапе из-за проблем с установкой зависимостей на Windows.

**Причина:**  
Установка `crewai` + `numpy` вызывает критические ошибки компиляции. Продолжение борьбы с инфраструктурой будет сильно тормозить развитие проекта.

**Альтернатива:**  
Использовать более лёгкий подход:
- Сохранить всю структуру `memory-bank`
- Вести агентов через промпты (Claude / Cursor / Grok)
- Постепенно возвращаться к автоматизации позже

**Статус:** Принято

## 16.07.2026 — Состав агентов

**Решение:**  
Зафиксировать состав из **3 агентов**:
- **Orchestrator** — главный координатор
- **CodeCritic** — критик кода и архитектуры
- **Improver** — генератор улучшений

**Причина:**  
Оптимальное соотношение между качеством работы и расходом токенов. Четыре агента избыточно на данном этапе.

**Статус:** Принято
