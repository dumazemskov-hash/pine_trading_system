# Активный контекст — Liquidity Raid Hunter (Multi-Agent System)

## Текущий фокус сессии
Настройка автоматизированной системы из 3 ИИ-агентов на базе **CrewAI** для глубокого анализа и улучшения существующего кода Liquidity Raid Hunter (сканер + индикатор).

Агенты общаются между собой через файлы в папке `memory-bank`.

## Состав агентов
- **Orchestrator** — главный координатор проекта. Управляет задачами, обновляет память, принимает ключевые решения.
- **CodeCritic** — критик кода и архитектуры. Ищет проблемы, баги, слабые места и технический долг.
- **Improver** — генератор улучшений. На основе критики предлагает и прорабатывает конкретные улучшения кода и логики.

## Приоритет
Главный приоритет — **глубокий анализ и улучшение существующего кода** (сканер + индикатор), а не генерация новых идей с нуля.

## Ближайшие задачи
1. Зафиксировать структуру `memory-bank`
2. Определить чёткие роли и промпты для каждого агента
3. Настроить базовый запуск CrewAI
4. Начать первый цикл анализа текущего кода
